# Simple User Authentication System

## Description
A simple web authentication project developed using Node.js and Express.js.
It includes signup, login, and dashboard modules.

## Features
- Signup using POST method
- Login using POST method
- Password hashing using bcrypt
- Email duplication prevention
- Simple user interface

## Technologies
- Node.js
- Express.js
- MySQL
- HTML, CSS

## Modules
- Signup
- Login
- Dashboard
